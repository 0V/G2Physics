# G2Physics
今までにやったなんちゃって物理シミュレーションを置く場所

## G2Physics
一次元 Schrodinger 方程式（定常状態）を解く。
* 時間に依存しないやつ解きます
* 4次の Runge-Kutta 法で解きます
* シューティング法を用います
* plotは別途gnuplotなどを使ってください
